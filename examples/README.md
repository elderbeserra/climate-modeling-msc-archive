# Examples and Legacy Tests

This folder keeps historical test/example scripts used during development.

- `teste.gs`: GrADS test snippet.
- `teste_dump.gs`: GrADS dump/quick-check script.
- `teste_readcsh.csh`: Shell test for reading/execution checks.
- `tutorial_fort.f90`: Fortran tutorial/prototype program.

These files are preserved for archive completeness and may not represent production-ready workflows.
